# Vercel Deployment Note

This repository contains a **React Native/Expo mobile application**, not a web application.

**Vercel cannot deploy this project** as it is designed for iOS and Android native apps.

## Deployment Options:

- **iOS**: Use EAS Build + App Store Connect
- **Android**: Use EAS Build + Google Play Console
- **Expo Go**: Publish with `expo publish` or EAS Update

See the main README.md for setup instructions.
